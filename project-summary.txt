DEPLOYMENT INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════

1. LOCAL SETUP
   ```bash
   git clone https://github.com/yourusername/iot-device-health-agents.git
   cd iot-device-health-agents
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env
   # Edit .env with your API keys
   ```

2. RUN LOCALLY
   ```bash
   python -m agents.root_agent
   ```

3. RUN EVALUATION
   ```bash
   python -m evaluation.eval_metrics
   ```

4. DEPLOY TO VERTEX AI
   ```bash
   python deployment/deploy_to_vertex.py
   ```
